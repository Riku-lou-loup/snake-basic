# Snake 64

Small Snake game for the Commodore 64, written in BASIC V2.

## Tooling

This project targets VICE, not QEMU. QEMU is installed on this machine, but it does not emulate a C64. VICE is already available here and includes:

- `x64sc` / `x64` for emulation
- `petcat` to tokenize BASIC text into a `.prg`
- `c1541` to build a `.d64` disk image
- `Xvfb` and `xset` to support headless smoke tests
- `bash`, `coreutils`, and `make` for the project scripts

## Nix Shell

This repository now includes `shell.nix` with the required tools.

Enter the environment with:

```bash
nix-shell
```

## Files

- `src/snake.bas`: editable BASIC source
- `build/snake64.prg`: tokenized C64 program generated by the build
- `build/snake64.d64`: disk image generated by the build
- `shell.nix`: reproducible development shell for the required tools
- `tools/build.sh`: build both artifacts
- `tools/run.sh`: launch the game in VICE
- `tools/smoke-test.sh`: short boot test under `Xvfb`

## Usage

```bash
make build
make run
make smoke
```

Or run the scripts directly:

```bash
./tools/build.sh
./tools/run.sh
./tools/smoke-test.sh
```

## Controls

- `W`: up
- `A`: left
- `S`: down
- `D`: right
- `Space`: start
- `R`: restart after game over
- `Q`: quit after game over

## Notes

- `tools/run.sh` uses the current graphical session if `DISPLAY` is set.
- If no display is present, it starts `Xvfb` automatically and launches VICE there.
- The smoke test only verifies that VICE can boot the generated program; it does not play the game automatically.
